Business Questions answered using SQL and Power BI

About the project:
This project explores sales data for an automobile manufacturer, using SQL and Power BI to uncover insights.
The analysis answers key business questions, highlights trends, and provides actionable recommendations.

SQL Responses to Business Questions

1. Which products should we order more of or less of?

This analysis helps optimize inventory levels to prevent overstocking or running out of stock.

2. What are the monthly sales trends over the past year?

SELECT MONTH(SaleDate) AS Month, SUM(Revenue) AS MonthlyRevenue
FROM Sales
WHERE SaleDate >= DATEADD(YEAR, -1, GETDATE())
GROUP BY MONTH(SaleDate)
ORDER BY Month;

3. Which regions contributed the most to sales revenue?

SELECT Region, SUM(Revenue) AS RegionalRevenue
FROM Sales
GROUP BY Region
ORDER BY RegionalRevenue DESC;
